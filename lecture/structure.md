# Theoretical Physics 2: Quantum Mechanics -- Lecture Script Structure

This project follows the lecture-script structure used in the Experimental Physics project.

## Files

- `main.tex`: main document
- `preamble/packages.tex`: package imports
- `preamble/commands.tex`: shared commands, including quantum-mechanics notation
- `preamble/boxes.tex`: shared tcolorbox environments
- `chapters/*.tex`: chapter files, currently intentionally empty
